# Export Notes

This folder is reserved for PNG/SVG exports of screens.

## Quick export workflow

1. Open `preview/index.html` in browser
2. Use full-page screenshot extension or Windows Snipping Tool
3. Save mockups here for Gumroad gallery images:
   - `dashboard-overview.png`
   - `web3-wallets.png`
   - `auth-login.png`
   - `pricing-page.png`

Recommended gallery size: **1600×1000** or **1920×1080**
