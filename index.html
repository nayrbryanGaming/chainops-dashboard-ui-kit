Design System

Tokens that travel
between tools

CSS Custom Properties
Every color, shadow, and radius value lives as a CSS variable in one place. Override at the root level to theme the entire system.
JSON Export for Figma
Drop design-tokens/tokens.json into Figma Variables or Style Dictionary to stay in sync with the coded components.
Responsive by Default
CSS Grid and clamp() scale every layout and type size. The sidebar collapses, cards stack, and nothing breaks below 400px.
