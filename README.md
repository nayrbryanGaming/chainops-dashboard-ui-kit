## Quick Start

1. Open **`preview/index.html`** in your browser (Preview Hub).
2. Browse all screens from the hub navigation.
3. Customize colors in `assets/css/chainops.css` (`:root` variables).
4. Copy HTML sections into React, Next.js, or your stack.

### Regenerate previews (optional)

```bash
python tools/generate_previews.py
```
